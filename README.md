Descripción general
Este proyecto analiza la relación entre la movilidad urbana (congestión vehicular y tiempos de viaje) y la productividad económica (PIB per cápita) en las principales ciudades de América Latina. El objetivo es identificar patrones, correlaciones y posibles inconsistencias en los datos.
Objetivo
•	Evaluar la relación entre los índices de congestión de tráfico y el PIB per cápita.
•	Identificar patrones y tendencias entre distintas ciudades.
•	Analizar la calidad y consistencia de las fuentes de datos externas.
Herramientas y tecnologías
•	Python
•	Jupyter Notebook
•	Librerías: pandas, numpy, matplotlib, seaborn
Metodología
•	Limpieza y validación de datos.
•	Análisis exploratorio de datos (EDA).
•	Análisis de correlación.
•	Comparación entre ciudades.
Principales hallazgos
•	Se observó una correlación positiva entre la congestión vehicular y el PIB per cápita, consistente con el fenómeno económico habitual donde una mayor actividad económica suele generar mayor congestión.
•	Ciudades como Ciudad de México presentan alta congestión y un PIB relativamente alto.
•	En el caso de Santiago de Chile, se identificó alta congestión con un PIB relativamente bajo, lo que sugiere posibles inconsistencias en la fuente de datos y la necesidad de un análisis más profundo.
Aprendizajes clave
•	Importancia de la validación de datos provenientes de fuentes externas.
•	Uso del análisis exploratorio para identificar patrones y anomalías.
•	Comunicación clara de resultados y limitaciones del análisis.
